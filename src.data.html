src.data.load_gdrive_file(fid: str, cls: Type[T] | None, data_dir: str = 'data/') → T | None¶

Download a file from Google Drive using its file ID, move it into a data directory, and optionally load it into a class.

Parameters:

  • fid – Google Drive file ID.

  • cls – Class to wrap the loaded numpy array in. If None, no class is returned.

  • data_dir – Target directory where the file will be stored. (default: “data/”).

Returns:

An instance of cls containing the loaded data, or None if cls is None.

src.data.load_s_curve(n_samples: int = 100, *, noise: float = 0.0, random_state: int | None = None) → Tuple[Points, Coordinates]¶

Generate a synthetic S-curve dataset.

Parameters:

  • n_samples – Number of samples to generate. (default: 100).

  • noise – Standard deviation of Gaussian noise added to the data. (default: 0.0).

  • random_state – Seed for reproducibility. (default: None).

Returns:

A tuple (Points, Coordinates) where Points are the 3D data and Coordinates are the unrolled 1D parameters.

src.data.load_swiss_roll(n_samples: int = 100, *, noise: float = 0.0, random_state: int | None = None, hole: bool | None = False) → Tuple[Points, Coordinates]¶

Generate a synthetic Swiss roll dataset.

Parameters:

  • n_samples – Number of samples to generate. (default: 100).

  • noise – Standard deviation of Gaussian noise added to the data. (default: 0.0).

  • random_state – Seed for reproducibility. (default: None).

  • hole – Whether to generate the Swiss roll with a hole. (default: False).

Returns:

A tuple (Points, Coordinates) where Points are the 3D data and Coordinates are the unrolled 1D parameters.
