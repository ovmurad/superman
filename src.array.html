class src.array.base.Array(array: Union[ndarray[tuple[Any, ...], dtype[_ScalarT]], csr_array])¶

Bases: ABC

Abstract base class for array-like objects (dense or sparse).

Wraps the core interface shared by numpy and scipy.sparse arrays, and defines an abstract contract for key operations that are format-specific.
